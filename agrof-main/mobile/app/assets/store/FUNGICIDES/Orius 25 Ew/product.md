# Orius 25 Ew

This is a Systemic / Curative fungicide with Tebuconazole 250g/L active ingredient. It is used to control fungal infections such as Leaf Rust, Net Blotch, Helminthosporium spots/ Blight, Septoria Leaf Spot.
